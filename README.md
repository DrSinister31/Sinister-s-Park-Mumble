# Sinister's Park Mumble

Proximity voice chat installer for The Isle: Evrima — Sinister's Park.

## Download

Get the latest installer from [Releases](https://github.com/Sinister-s-Park/Sinister-s-Park-Mumble/releases/latest).

## What It Does

- Installs/configures Mumble 1.5.901
- Pre-connects to `sinisterspark.duckdns.org:64738`
- Offers username setup during install

## After Install — Link Your Voice

1. Join The Isle: Evrima — Sinister's Park
2. Type `!verify` in game chat *(or ask SPS in Discord for a code)*
3. Check your Discord DMs — SPS bot sends a 6-digit code
4. In Mumble chat, type `!verify <code>`
5. Done — proximity voice auto-sorts your channel by in-game location
