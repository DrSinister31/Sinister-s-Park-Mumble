# Sinister's Park Mumble

Proximity voice chat installer for The Isle: Evrima — Sinister's Park.

## Download

Get the latest installer from [Releases](https://github.com/DrSinister31/Sinister-s-Park-Mumble/releases/latest).

## What It Does

- Installs/configures Mumble 1.5.901
- Pre-connects to `sinisterspark.duckdns.org:64738`
- Offers username setup during install

## After Install — Link Your Voice

1. Join The Isle: Evrima — Sinister's Park
2. Ask SPS in Discord: "verify my voice"
3. Check your Discord DMs — SPS bot sends a 6-digit code
4. In Mumble chat, type `!verify <code>`
5. Done — proximity voice auto-sorts your channel by in-game location

## Manual Setup (no installer)

1. Open Mumble → Server → Connect → Add New
2. Address: `sinisterspark.duckdns.org`, Port: `64738`
3. Username: your in-game name
4. Click Connect

## Download Blocked or Flagged as Unsafe?

This is normal for unsigned software. Bypass:

**Edge:** Click `...` on download bar → `Keep` → `Show more` → `Keep anyway`

**Chrome:** Click download arrow → `Keep` or `Keep dangerous file`

**Firefox:** Click download arrow → right-click the file → `Allow download`

**Windows SmartScreen (when running):** Click `More info` → `Run anyway`

## Uninstall

Control Panel → "Sinister's Park Mumble" → Uninstall. Mumble and your other plugins stay untouched.
