# Manual Mumble Setup — Sinister's Park

No installer needed. 3 steps.

## 1. Install Mumble

Download from: https://www.mumble.info/downloads/

## 2. Add the server

- Open Mumble
- Server → Connect → **Add New...**
- Label: `Sinister's Park`
- Address: `sinisterspark.duckdns.org`
- Port: `64738`
- Username: **your in-game name** (same as The Isle)

## 3. Click Connect

Done. Push-to-talk key is `T`.

## Link your voice (one-time)

1. Join the game
2. Ask SPS on Discord: "verify my voice"
3. Check your Discord DMs for a code
4. Type `!verify <code>` in Mumble chat
5. Done — voice auto-sorts by location

## Uninstall

Control Panel → Uninstall "Sinister's Park Mumble" (if you used the installer).
If you set up manually, just remove the server entry in Mumble's Server list.
Your Mumble program and other plugins stay untouched.

## Reinstall / Reset

Just run the installer again. It only reconfigures Sinister's Park settings.
Or delete the server entry and re-add it manually.

## Troubleshooting

- No code in DMs? Use `/link` in Discord first, then ask SPS again.
- Can't hear? In-game + verified = required.
- Wrong name? Disconnect → edit username → reconnect.
- Plugin needed? No. Channel sorting is server-side.
